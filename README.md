# Student Grade Management System

## Description

This project is a simple Student Grade Management System written in C++.

The project demonstrates fundamental data structures and file handling concepts.

## Features

- Add Student
- Delete Student
- Search Student
- Display Students
- Save Students to File
- Load Students from File
- Linked List Implementation
- Bitwise Flags for Student Status

## Technologies

- C++
- Struct
- Linked List
- File Handling
- Bitwise Operations
- Makefile
